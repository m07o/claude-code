# Acknowledgements

This project stands on the shoulders of outstanding open-source projects and contributions from the community.

## Core Projects

### **Anthropic - Claude Code**
The original Claude Code CLI and framework that made this project possible.
- **Repository**: https://github.com/anthropics/claude-code
- **License**: See original project
- **Used for**: Base CLI implementation, protocol specifications, core functionality

### **Groq - Fast LLM Inference**
High-performance LLM inference API enabling rapid model serving.
- **Website**: https://www.groq.com/
- **API**: https://console.groq.com/
- **Used for**: Primary LLM backend, high-speed inference

### **Meta - Llama Models**
State-of-the-art open-source language models that power the proxy.
- **Llama 4 Scout 17B**: Best performance for code tasks
- **Llama 3.3 70B**: Balanced capability and speed
- **Llama 3.1 8B**: Fast, efficient, cost-effective
- **Repository**: https://github.com/meta-llama/llama
- **License**: Llama 2 Community License Agreement

### **Alibaba - Qwen Models**
Advanced language models providing alternative inference paths.
- **Qwen3 32B**: High-capacity multilingual model
- **Website**: https://huggingface.co/Qwen
- **Used for**: Alternative model option on Groq API

## Supporting Technologies

### **Ollama - Local LLM Inference**
Simple, fast local LLM serving for privacy and offline capabilities.
- **Website**: https://ollama.ai/
- **Repository**: https://github.com/ollama/ollama
- **Used for**: Local model support, testing, offline usage

### **OpenAI - Chat Completions API**
Reference implementation for the API format we translate to.
- **Documentation**: https://platform.openai.com/docs/api-reference/chat/create
- **Used for**: Protocol specification, compatibility layer

### **Node.js - JavaScript Runtime**
The foundation for our proxy server and CLI utilities.
- **Website**: https://nodejs.org/
- **License**: MIT
- **Used for**: Server runtime, batch scripts, development tools
